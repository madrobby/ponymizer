You know, like "anonymizer" but with ponies. Add `gem "ponymizer"` to your ~~Ponyfile~~ Gemfile.

```ruby
Ponymizer.new.gimme_a_pony!
# => "Sunshine Splash"
```

Ponymizer works with Ruby 1.8.7 or later.
